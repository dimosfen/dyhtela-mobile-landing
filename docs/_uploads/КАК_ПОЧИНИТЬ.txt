Ваши три файла лежат здесь как 1.png, 2.png, 3.png.

Соответствие «какой хеш Canva → какой файл» задаётся в файле
  ../_canva-media-overrides.json
(в папке public/, рядом с index.html).

Ключ — точное имя файла из выгрузки Canva, значение — 1.png / 2.png / 3.png.

Список имён из макета: в корне проекта visual-mock-her-design выполните
  npm run list-media

Если на главной «не та» картинка — поправьте ключи в JSON.
